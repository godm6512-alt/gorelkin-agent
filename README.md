# jarvis-architect

Шаблон архитектуры личного Агента в Claude Code. Часть курса **«Архитектор нейросотрудников»** Дмитрия Ледовских.

## Что это такое

Готовый набор файлов, из которых собирается ваш персональный Агент. Архитектура — три уровня памяти:

**Главные файлы Агента (всегда читаются)**

| Файл | Что внутри |
|---|---|
| `CLAUDE.md` | Правила работы Агента в этой папке. Claude Code читает первым. |
| `SOUL.md` | Личность Агента: тон, ценности, что любит / не любит. |
| `MEMORY.md` | Долгосрочная память: профиль, проекты, инфраструктура, стек, предпочтения. |
| `GOALS.md` | Цели на квартал и активные направления. |

**Папки рабочей памяти (создаются и наполняются по ходу)**

| Папка | Что внутри |
|---|---|
| `memory/` | Дневники работы по дням (`YYYY-MM-DD.md`). Агент ведёт сам. |
| `knowledge/` | Справочники и инструкции (брендбук, чек-листы, гайды). Вы кладёте сюда вручную или просите Агента собрать. |

**Только в шаблоне (удаляется после установки)**

| Файл / папка | Назначение |
|---|---|
| `INSTALL.md` | Wizard-инструкция для Claude Code |
| `examples/` | Заполненный пример Агента (для подсматривания) |

Все 4 главных файла приходят сразу с плейсхолдерами вроде `{{NAME}}`, `{{ROLE}}` — они заполняются в процессе короткого интервью с Claude Code.

## Как установить (Урок 4)

В вашем Claude Code, открытом в папке `jarvis/`, скажите одну фразу:

> «Установи мне архитектуру Агента из github.com/Ntmib/jarvis-architect»

Дальше Claude сам:

1. Скачает шаблон в текущую папку через `npx degit Ntmib/jarvis-architect .`
2. Прочитает `INSTALL.md` — там пошаговая инструкция для него
3. Задаст вам 10 вопросов: имя, чем занимаетесь, цели, стиль общения, что бесит, какой Агент Вам нужен
4. Заменит плейсхолдеры в файлах вашими ответами (файлы остаются в корне, никуда не двигаются)
5. Покажет вам превью каждого файла — Вы подтверждаете или просите поправить
6. Удалит служебное (`INSTALL.md`, `README.md`, `dna-map/`, `examples/`)
7. Сделает первый коммит + push в ваш GitHub-репозиторий

Время от команды до готовой архитектуры — 10–12 минут.

## Если предпочитаете команду в терминале

```bash
npx degit Ntmib/jarvis-architect .
```

После этого откройте Claude Code в этой папке — он сам найдёт `INSTALL.md` и продолжит установку.

## Если что-то пошло не так

В папке `examples/` лежат заполненные примеры — подсмотрите, как должны выглядеть готовые файлы. Если интервью с Claude сбилось, можно скопировать пример и поправить под себя.
